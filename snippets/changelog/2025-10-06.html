Platform Simplification, New Conversational Metrics, and Major Usability Upgrades

2025-10-06

Platform Simplification and SDK v3.0

We've simplified our core concepts to make the platform more intuitive. For example, `Evaluation Tasks` are now simply Evaluations. These changes come with a new major SDK version. Please see the Migration to v3.0 guide for a smooth transition.


New Conversational Metrics

We're excited to introduce two new metrics for evaluating conversational AI: User Objective Accomplished and User Satisfaction.


Enhanced Test Case Feedback

You can now add a "user score reason" when upvoting or downvoting a Test Case to provide more context. You can also filter by score in the SDK.


Dashboard & SDK Usability Upgrades

We've rolled out several updates to make your workflow smoother. In the Dashboard, you can now single-click table rows to copy text, and right-click to navigate to details. In the SDK, you can now fetch objects by passing a list of IDs, making batch operations much easier.


General Improvements

This release also includes numerous performance optimizations and UI/UX enhancements across the entire platform to provide a faster and more polished experience.
