Platform Simplification, New Metrics, Scenarios & Performance Boost

2025-10-06

Platform Simplification and SDK v3.0

We've simplified core concepts to make the platform more intuitive. For example, `Evaluation Tasks` are now simply Evaluations. These changes come with a major SDK release — see the Migration to v3.0 guide for details.


New Conversational Metrics

Introducing two conversational metrics: User Objective Accomplished and User Satisfaction, to better capture multi-turn success and perceived quality.


Generate Scenarios from Quality Tests

You can now create multi-turn Scenario Based Tests directly from existing Quality Tests, turning single-turn gold tests into realistic conversational flows for deeper evaluation.


URL Validation & Deterministic Checks

Added a deterministic URL Validation metric that verifies HTTPS, formatting, resolvability and includes SSRF protections — useful when models emit external links.


Enhanced Test Case Feedback & SDK Filters

You can now add a "user score reason" when voting on a Test Case, and the SDK supports filtering by score and fetching objects in batches (pass a list of IDs).


Dashboard Usability & Performance

Dashboard UX improvements include one-click copy of table rows and right-click navigation. We also shipped platform-wide performance boosts, especially in analytics views, for a snappier experience.
