Test Case Confidence, Custom Judges for Conversations, and More

2025-09-15

Confidence Scores for Generated Test Cases

We're introducing confidence scores for all generated Test Cases. This provides a clear indicator of test case quality, helping you prioritize human review and better understand your test suites.


Simplified and More Flexible Metric Creation

Creating Metric Types is now more intuitive. Metrics are now directly linked to a specific test type (QUALITY, RED_TEAMING, or SCENARIOS), simplifying the creation process by tailoring the available parameters to the relevant test category. See the updated creation guide for more details.


Custom Judges for Conversational Evaluation

You can now create your own Custom Judges specifically for evaluating multi-turn conversations. This allows you to define complex evaluation logic to assess nuanced aspects like task completion, context retention, and persona adherence across an entire dialogue.


Conversation Simulator Enhancements

We've added more control and realism to the Conversation Simulator:

  • Agent-First Interactions: A new agent_goes_first parameter lets you test scenarios where the agent initiates the conversation.
  • Selectable Conversation Styles: Choose between written and spoken styles when creating tests to influence the tone and formality of the synthetic user's dialogue.
