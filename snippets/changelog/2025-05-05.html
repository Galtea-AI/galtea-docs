Analytics Upgrades and Red Teaming Test Improvements

2025-05-05

Analytics Page Upgrades

We're continuing to improve the Analytics page to help you extract insights faster:

  • Radar View for Version Comparison: Visualize performance across multiple metrics with our new radar view.
  • Smarter Metric Filters: Only shows metrics used in evaluations—less clutter, more focus.
  • Graph Tooltips: Hover to reveal truncated names inside charts.

SDK Safeguards

We've introduced two improvements to make SDK usage more reliable:

  • Version Compatibility Checks: Clear errors when the SDK version is incompatible with the API.
  • Update Notifications: Get notified in the console when a newer SDK version is available.

Improvements in Red Teaming Tests

  • New "misuse" threat implemented:
    Now red teaming incorporates a new threat, misuse, which are queries that not necessaryly malicious however out-of-scope for you specific product. You can now test whether your product can successfully block these queries by marking "Mitre Atlas: Ambiguous prompts" in the threat list.
  • Better "data leakage" and "toxicity" tests:
    The red teaming tests incorporate better your product meta data, to generate the most adequate test cases for "data leakage" and "toxicity".

Bug Fixes

  • Metric Range Fix: Default metrics now correctly show 100% as best and 0% as worst.

We hope these upgrades make your experience better and more efficient.
