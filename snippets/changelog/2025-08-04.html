Introducing Custom Metrics: Define Your Own Success

You can now create fully customized evaluation metrics using natural language prompts. This powerful feature empowers you to define precisely what "good" looks like for your specific use case, going beyond standard metrics to test for nuanced behaviors, specific formats, or brand voice compliance. Learn how to create your own.


Expanded Library of Built-in Metrics

We've also expanded our library of classical metrics to give you more out-of-the-box evaluation power. This release includes:

  • Text Similarity & Text Match: Measure semantic similarity or check for exact string matches.
  • Spatial Match: Verify if generated locations fall within a defined area.
  • Intersection over Union (IoU): A crucial metric for object detection and computer vision tasks.

Explore all available metrics in our documentation.
