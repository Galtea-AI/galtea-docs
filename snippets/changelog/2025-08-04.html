New Deterministic Metrics

Four new deterministic metrics are now available, perfect for scenarios requiring rule-based scoring:

  • Text Similarity: Quantifies text resemblance using character-level fuzzy matching.
  • Text Match: Returns pass/fail based on similarity to a reference string.
  • Spatial Match: Verifies box alignment with IoU scoring for a pass/fail result.
  • IoU (Intersection over Union): Computes the overlap ratio between predicted and reference boxes.
