New Evaluator Models, Classic Metrics, and Dashboard Redesign

2025-07-28

Expanded Evaluator Model Support

We've added support for more evaluator models: Gemini-2.5-Flash, Gemini-2.5-Flash-Lite, and Gemini-2.0-Flash. You can now link these models to specific Metrics for consistent evaluation results and leverage specialized models for different scenarios.

Enhanced Conversation Simulation

Testing conversational AI just got more powerful with two major improvements:

  • Visible Stopping Reasons: See exactly why simulated conversations ended in the dashboard for crucial insights into dialogue flow.
  • Custom User Persona Definitions: Create highly specific synthetic user personas when generating Scenario Based Tests to test diverse user interactions.

Classic NLP Metrics Now Available

We've expanded our metric library with three essential deterministic metrics:

  • BLEU: Measures n-gram overlap for machine translation and constrained generation tasks.
  • ROUGE: Evaluates summarization quality by measuring longest common subsequences.
  • METEOR: Assesses translation and paraphrasing using exact matches, stems, and synonyms.

Enhanced Red Teaming with Jailbreak Resilience v2

Security testing gets an upgrade with Jailbreak Resilience v2, an improved version of our jailbreak resistance metric that provides more comprehensive assessment of your model's ability to resist adversarial prompts and maintain safety boundaries.

Dashboard Redesign: First Iteration

We've launched the first iteration of our redesigned dashboard with a refreshed visual language focused on clarity and usability:

  • Modern Typography: Cleaner, more readable text throughout the platform.
  • Refined UI Elements: Updated buttons, cards, and forms with a more contemporary look.
  • Streamlined Tables: Enhanced data presentation with improved content layout.
  • Updated Login Experience: A more polished and user-friendly authentication flow.

Improved SDK Documentation

We've enhanced our SDK documentation with clearer guidance on defining evaluator models for metrics, making it easier to configure and customize your evaluation workflows.
