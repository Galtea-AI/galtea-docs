---
title: 'Retrieving Evaluation Task'
description: "Retrieve a specific evaluation task by its ID."
---

## Returns
Returns an [EvaluationTask](/concepts/product/version/session/evaluation-task) object for the given task ID.

## Example
```python
evaluation_task = galtea.evaluation_tasks.get(evaluation_task_id="YOUR_TASK_ID")
```

## Parameters
<ResponseField name="evaluation_task_id" type="string" required>
  The ID of the task you want to retrieve.
</ResponseField>
