---
title: 'Get Session'
description: "Retrieve a specific session by its ID."
---

## Returns
Returns a [Session](/concepts/product/session) object.

## Example
```python
session = galtea.sessions.get(session_id="YOUR_SESSION_ID")
```

## Parameters
<ResponseField name="session_id" type="string" required>
  The ID of the session to retrieve.
</ResponseField>
