---
title: 'List Sessions'
description: "List all sessions for a product or version"
---

List all [sessions](/concepts/product/session) with optional filtering by version or other criteria.

## Parameters

<ParamField path="version_id" type="string" optional>
  Filter sessions by version ID. If not provided, returns sessions for all versions.
</ParamField>

<ParamField path="test_case_id" type="string" optional>
  Filter sessions by test case ID.
</ParamField>

<ParamField path="limit" type="number" optional default="50">
  Maximum number of sessions to return (1-100).
</ParamField>

<ParamField path="offset" type="number" optional default="0">
  Number of sessions to skip for pagination.
</ParamField>

## Returns

Returns a list of [Session](/concepts/product/session) objects.

## Example

```python
# List all sessions for a specific version
sessions = galtea.sessions.list(
    version_id="YOUR_VERSION_ID",
    limit=50,
    offset=0
)
```
