---
title: 'Create Session'
description: "Create a new session to group inference results."
---

## Returns
Returns a [Session](/concepts/session) object.

## Example
```python
session = galtea.sessions.create(
    version_id="YOUR_VERSION_ID",
    test_case_id="YOUR_TEST_CASE_ID", # Optional, for non-production
    is_production=False
)
```

## Parameters
<ResponseField name="version_id" type="string" required>
  The ID of the version to associate with this session.
</ResponseField>
<ResponseField name="id" type="string" optional>
  A client-provided session ID. If not provided, one will be auto-generated.
</ResponseField>
<ResponseField name="test_case_id" type="string" optional>
  The ID of a test case. Required for non-production sessions (`is_production=False`).
</ResponseField>
<ResponseField name="context" type="dict" optional>
  A flexible key-value store for user-defined context.
</ResponseField>
<ResponseField name="is_production" type="boolean" optional>
  A flag to indicate if the session is from a production environment. Defaults to `False`.
</ResponseField>
