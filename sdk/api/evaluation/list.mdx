---
title: 'Listing Evaluations'
description: "List all evaluations associated with a specific session."
---


## Returns
Returns a list of [Evaluation](/concepts/product/version/session/evaluation) objects.

## Example
```python
# List tasks for a session
session_evaluations = galtea.evaluations.list(session_id="YOUR_SESSION_ID")
```

## Parameters
<ResponseField name="session_id" type="string" required>
  The ID of the session for which you want to list tasks.
</ResponseField>
<ResponseField name="offset" type="int">
  The number of tasks to skip before starting to collect the result set.
</ResponseField>
<ResponseField name="limit" type="int">
  The maximum number of tasks to return.
</ResponseField>
