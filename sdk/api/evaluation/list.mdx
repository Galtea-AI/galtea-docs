---
title: 'Listing Evaluations'
description: "List all evaluations associated with a specific product."
---

## Returns
Returns a list of [Evaluation](/concepts/product/evaluation) objects for the given product.

## Example
```python
evaluations = galtea.evaluations.list(product_id="YOUR_PRODUCT_ID")
```

## Parameters
<ResponseField name="session_id" type="string" required>
  The ID of the session for which you want to list evaluations.
</ResponseField>
<ResponseField name="offset" type="int">
  The number of evaluations to skip before starting to collect the result set.
</ResponseField>
<ResponseField name="limit" type="int">
  The maximum number of evaluations to return.
</ResponseField>
