---
title: 'Deleting Evaluation Task'
description: "Delete a specific evaluation task by its ID."
---

## Example
```python
galtea.evaluation_tasks.delete(evaluation_task_id="YOUR_TASK_ID")
```

## Parameters
<ResponseField name="evaluation_task_id" type="string" required>
  The ID of the task you want to delete.
</ResponseField>
