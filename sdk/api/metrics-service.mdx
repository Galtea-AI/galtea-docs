The Metrics Service in the Galtea SDK allows you to manage metrics for evaluating your products.
This Service is exposed by the `galtea.metrics` object and we will further explore its API down below.

Remember that we will be using the `galtea` object, which is an instance of the `Galtea` class.
```python
from galtea import Galtea
galtea = Galtea(api_key="<YOUR_GALTEA_PLATFORM_API_KEY>")
```

## Create Metric

This method allows you to create a metric.

### Parameters

- `name`
  - The name of the metric.
  - REQUIRED: yes
- `criteria`
  - The criteria for the metric.
  - REQUIRED: yes

### Example Code

```python
metric = galtea.metrics.create(
    name="accuracy_v0",
    criteria="Determine whether the actual output is equivalent to the expected output."
)
print(metric.id)
```

## Listing Metrics

This method allows you to list all metrics.

### Example Code

```python
metrics = galtea.metrics.list()
for metric in metrics:
    print(metric.name, metric.id)
```

## Retrieving Metric

This method allows you to retrieve a specific metric by its ID.

### Parameters

- `metric_id`
  - The ID of the metric you want to retrieve.
  - REQUIRED: yes

### Example Code

```python
metric = galtea.metrics.get(metric_id="YOUR_METRIC_ID")
print(metric.name, metric.criteria)
```

## Updating Metric

This method allows you to update a specific metric by its ID.

### Parameters

- `metric_id`
  - The ID of the metric you want to update.
  - REQUIRED: yes
- `name`
  - The new name of the metric.
  - REQUIRED: no
- `optional_props`
  - Additional properties for the metric (e.g., criteria).
  - REQUIRED: no

### Example Code

```python
updated_metric = galtea.metrics.update(
    metric_id="YOUR_METRIC_ID",
    name="updated-metric-name",
    optional_props={"criteria": "Updated criteria description"}
)
print(updated_metric.name, updated_metric.criteria)
```

## Deleting Metric

This method allows you to delete a specific metric by its ID.

### Parameters

- `metric_id`
  - The ID of the metric you want to delete.
  - REQUIRED: yes

### Example Code

```python
galtea.metrics.delete(metric_id="YOUR_METRIC_ID")
print("Metric deleted successfully")
```
