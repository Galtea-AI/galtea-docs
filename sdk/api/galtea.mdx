## Galtea

The `Galtea` class is the main entry point for the Galtea SDK. It provides access to all the functionalities of the SDK.

### Initialization

To initialize the `Galtea` class, you need to provide your API key.

```python
from galtea import Galtea

# Initialize Galtea SDK
galtea = Galtea(api_key="YOUR_API_KEY")
```

### Methods

The `Galtea` class provides the following methods:

- `products`: Access product-related functionalities.
- `versions`: Access version-related functionalities.
- `tests`: Access test-related functionalities.
- `metrics`: Access metric-related functionalities.
- `evaluations`: Access evaluation-related functionalities.
- `evaluation_tasks`: Access evaluation task-related functionalities.

### Example

```python
from galtea import Galtea

# Initialize Galtea SDK
galtea = Galtea(api_key="YOUR_API_KEY")

# List products
products = galtea.products.list()
print(products)
```
