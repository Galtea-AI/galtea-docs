---
title: 'Delete Product'
description: "Delete a specific product by its ID."
---

## Returns

Returns a deletion confirmation object and the deleted [Product](/concepts/product/product) object.

## Example
```python
galtea.products.delete(product_id="YOUR_PRODUCT_ID")
```

## Parameters
<ResponseField name="product_id" type="string" required>
  The ID of the product you want to delete.
</ResponseField>
