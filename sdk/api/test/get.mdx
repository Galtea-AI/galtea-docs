---
title: 'Retrieve Test'
description: "Retrieve a specific test by its ID."
---

## Returns
Returns a [Test](/concepts/product/test) object.

## Example
```python
test = galtea.tests.get(test_id="YOUR_TEST_ID")
```

## Parameters
<ResponseField name="test_id" type="string" required>
  The ID of the test you want to retrieve.
</ResponseField>
