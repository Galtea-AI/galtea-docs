---
title: 'Delete Test'
description: "Delete a specific test by its ID."
---

## Returns

Returns a deletion confirmation object and the deleted [Test](/concepts/product/test) object.

## Example
```python
galtea.tests.delete(test_id="YOUR_TEST_ID")
```

## Parameters
<ResponseField name="test_id" type="string" required>
  The ID of the test you want to delete.
</ResponseField>
