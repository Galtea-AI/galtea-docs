---
title: 'Retrieving Metric'
description: "Retrieve a specific metric by its ID."
---

## Returns
Returns a [Metric](/concepts/metric) object for the given metric ID.

## Example
```python
metric = galtea.metrics.get(metric_id="YOUR_METRIC_ID")
```

## Parameters
<ResponseField name="metric_id" type="string" required>
  The ID of the metric you want to retrieve.
</ResponseField>
