---
title: 'Retrieve Metric'
description: "Retrieve a specific metric by its ID."
---

## Example
```python
metric = galtea.metrics.get(metric_type_id="YOUR_METRIC_ID")
```

## Parameters
<ResponseField name="metric_type_id" type="string" required>
  The ID of the metric you want to retrieve.
</ResponseField>

## Returns

Returns a [MetricType](/concepts/metric-type) object.
