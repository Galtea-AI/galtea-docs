The Evaluation Task Service in the Galtea SDK allows you to manage evaluation tasks for assessing your products.
This Service is exposed by the `galtea.evaluation_tasks` object and we will further explore its API down below.

Remember that we will be using the `galtea` object, which is an instance of the `Galtea` class.
```python
from galtea import Galtea
galtea = Galtea(api_key="<YOUR_GALTEA_PLATFORM_API_KEY>")
```

## Create Evaluation Task

This method allows you to create an evaluation task.

### Parameters

<ResponseField name="evaluation_id" type="string" required>
  The ID of the evaluation you want to create a task for.
</ResponseField>

<ResponseField name="metric_type" type="string" required>
  The type of metric to use for the evaluation.
</ResponseField>

<ResponseField name="input" type="string" required>
  The input data for the evaluation task.
</ResponseField>

<ResponseField name="actual_output" type="string" required>
  The actual output produced by the product.
</ResponseField>

<ResponseField name="expected_output" type="string">
  The expected output for the evaluation task.
</ResponseField>

### Example Code

```python
evaluation_task = galtea.evaluation_tasks.create(
    evaluation_id="YOUR_EVALUATION_ID",
    metric_type="accuracy_v0",
    input="What is the capital of Spain?",
    actual_output="Barcelona",
    expected_output="Madrid"
)
print(evaluation_task.id)
```

## Listing Evaluation Tasks

This method allows you to list all evaluation tasks associated with a specific evaluation.

### Parameters

<ResponseField name="evaluation_id" type="string" required>
  The ID of the evaluation for which you want to list tasks.
</ResponseField>

### Example Code

```python
evaluation_tasks = galtea.evaluation_tasks.list(evaluation_id="YOUR_EVALUATION_ID")
for task in evaluation_tasks:
    print(task.id, task.status)
```

## Retrieving Evaluation Task

This method allows you to retrieve a specific evaluation task by its ID.

### Parameters

<ResponseField name="task_id" type="string" required>
  The ID of the task you want to retrieve.
</ResponseField>

### Example Code

```python
evaluation_task = galtea.evaluation_tasks.get(task_id="YOUR_TASK_ID")
print(evaluation_task.id, evaluation_task.status, evaluation_task.score)
```

## Updating Evaluation Task

This method allows you to update a specific evaluation task by its ID.

### Parameters

<ResponseField name="task_id" type="string" required>
  The ID of the task you want to update.
</ResponseField>

<ResponseField name="actual_output" type="string">
  The new actual output for the task.
</ResponseField>

### Example Code

```python
updated_task = galtea.evaluation_tasks.update(
    task_id="YOUR_TASK_ID",
    actual_output="Updated output"
)
print(updated_task.id, updated_task.actual_output)
```

## Deleting Evaluation Task

This method allows you to delete a specific evaluation task by its ID.

### Parameters

<ResponseField name="task_id" type="string" required>
  The ID of the task you want to delete.
</ResponseField>

### Example Code

```python
galtea.evaluation_tasks.delete(task_id="YOUR_TASK_ID")
print("Evaluation task deleted successfully")
```
