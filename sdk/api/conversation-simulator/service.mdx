The [Conversation Simulator](/concepts/conversation-simulator) Service in the Galtea SDK allows you to generate synthetic user messages to test conversational AI products.
This Service is exposed by the `galtea.simulator` object.

<Info>
  Remember that we will be using the `galtea` object. More information [here](/sdk/api/galtea).
</Info>
