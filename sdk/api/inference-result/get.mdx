---
title: 'Get Inference Result'
description: "Retrieve an inference result log by its ID."
---

## Returns
Returns an [InferenceResult](//concepts/product/inference-result) object.

## Example
```python
inference_result = galtea.inference_results.get(inference_result_id="YOUR_INFERENCE_RESULT_ID")
```

## Parameters
<ResponseField name="inference_result_id" type="string" required>
  The ID of the inference result log to retrieve.
</ResponseField>
