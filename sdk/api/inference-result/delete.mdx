---
title: 'Delete Inference Result'
description: "Delete an inference result log by its ID."
---

## Example
```python
galtea.inference_results.delete(inference_result_id="YOUR_INFERENCE_RESULT_ID")
```

## Parameters
<ResponseField name="inference_result_id" type="string" required>
  The ID of the inference result log to delete.
</ResponseField>
