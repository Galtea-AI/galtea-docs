The Inference Result Service in the Galtea SDK allows you to manage [inference results](//concepts/product/inference-result) for evaluating your products.
This Service is exposed by the `galtea.inference_results` object.

<Info>
  Remember that we will be using the `galtea` object. More information [here](/sdk/api/galtea).
</Info>
