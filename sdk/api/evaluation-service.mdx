The Evaluation Service in the Galtea SDK allows you to manage evaluations for assessing your products.
This Service is exposed by the `galtea.evaluations` object and we will further explore its API down below.

Remember that we will be using the `galtea` object, which is an instance of the `Galtea` class.
```python
from galtea import Galtea
galtea = Galtea(api_key="<YOUR_GALTEA_PLATFORM_API_KEY>")
```

## Create Evaluation

This method allows you to create an evaluation.

### Parameters

- `test_id`
  - The ID of the test you want to use for the evaluation.
  - REQUIRED: yes
- `version_id`
  - The ID of the version you want to evaluate.
  - REQUIRED: yes

### Example Code

```python
evaluation = galtea.evaluations.create(
    test_id="YOUR_TEST_ID",
    version_id="YOUR_VERSION_ID"
)
print(evaluation.id)
```

## Listing Evaluations

This method allows you to list all evaluations associated with a specific product.

### Parameters

- `product_id`
  - The ID of the product for which you want to list evaluations.
  - REQUIRED: yes

### Example Code

```python
evaluations = galtea.evaluations.list(product_id="YOUR_PRODUCT_ID")
for evaluation in evaluations:
    print(evaluation.id, evaluation.status)
```

## Retrieving Evaluation

This method allows you to retrieve a specific evaluation by its ID.

### Parameters

- `evaluation_id`
  - The ID of the evaluation you want to retrieve.
  - REQUIRED: yes

### Example Code

```python
evaluation = galtea.evaluations.get(evaluation_id="YOUR_EVALUATION_ID")
print(evaluation.id, evaluation.status)
```

## Updating Evaluation

This method allows you to update a specific evaluation by its ID.

### Parameters

- `evaluation_id`
  - The ID of the evaluation you want to update.
  - REQUIRED: yes
- `status`
  - The new status of the evaluation.
  - REQUIRED: no

### Example Code

```python
updated_evaluation = galtea.evaluations.update(
    evaluation_id="YOUR_EVALUATION_ID",
    status="completed"
)
print(updated_evaluation.id, updated_evaluation.status)
```

## Deleting Evaluation

This method allows you to delete a specific evaluation by its ID.

### Parameters

- `evaluation_id`
  - The ID of the evaluation you want to delete.
  - REQUIRED: yes

### Example Code

```python
galtea.evaluations.delete(evaluation_id="YOUR_EVALUATION_ID")
print("Evaluation deleted successfully")
```
