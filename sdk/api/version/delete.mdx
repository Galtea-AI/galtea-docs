---
title: 'Delete Version'
description: "Delete a specific version by its ID."
---

## Returns

Returns a deletion confirmation object and the deleted [Version](/concepts/product/version) object.

## Example
```python
galtea.versions.delete(version_id="YOUR_VERSION_ID")
```

## Parameters
<ResponseField name="version_id" type="string" required>
  The ID of the version you want to delete.
</ResponseField>
