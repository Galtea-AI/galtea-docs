---
title: 'Retrieve Version'
description: "Retrieve a specific version by its ID."
---

## Returns
Returns a [Version](/concepts/product/version) object.

## Example
```python
version = galtea.versions.get(version_id="YOUR_VERSION_ID")
```

## Parameters
<ResponseField name="version_id" type="string" required>
  The ID of the version you want to retrieve.
</ResponseField>
